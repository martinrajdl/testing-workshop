Why should I write tests?

Development is a complex process
And it's really hard to get things right, on the first try
You do something, then you do it better, then you do it better again
  • Test automation, fewer bugs, happier users
  • You make new things faster, happier project manager, stakeholder
  • Documenting code, happier colleagues
  • Code refactoring, happier you
Tests help you iterate, refactor, write better code
